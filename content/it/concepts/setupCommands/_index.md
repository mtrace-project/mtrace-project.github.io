## Introduzione
In questa sezione vengono descritti i comandi di setup di **Mtracer**. Questi comandi sono progettati per definire come preparare l'ambiente prima dell'esecuzione dei test e come ripulirlo al termine.
Nei file di test è possibile specificare un numero arbitrario di comandi di setup, che verranno eseguiti in ordine sequenziale prima e dopo l'esecuzione del test.

## Comandi di setup supportati
Attualmente, **Mtracer** supporta i comandi di setup illustrati nelle sezioni seguenti.   
Inoltre, il tipo di comando di setup di default è `shell`.
