## Introduzione
In questa sezione vengono descritti i comandi di setup di **Mtrace**. Questi comandi sono progettati per definire come preparare l'ambiente di test prima dell'esecuzione dei test e come pulirlo al termine dello stesso.    
Nei file di test è possibile specificare un numero qualsiasi di comandi di setup, che verranno eseguiti in ordine sequenziale prima e dopo l'esecuzione del test.

## Comandi di setup supportati
Attualmente, **Mtrace** supporta i comandi di setup illustrati nelle sezioni seguenti.   
Inoltre, il tipo di comando di setup di default è `shell`.
